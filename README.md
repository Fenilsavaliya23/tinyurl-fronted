# TinyURL Frontend

A modern React-based frontend for the TinyURL Platform that allows users to securely manage shortened URLs through an intuitive dashboard.

## Features

### Authentication

* User Registration
* User Login
* JWT Token Authentication
* Protected Routes
* Secure Session Management

### URL Management

* Create Short URLs
* Custom Alias Support
* Edit Existing Aliases
* Delete URLs
* URL Ownership Management

### Analytics

* Click Tracking
* Analytics Dashboard
* URL Statistics
* Expiration Monitoring

### Additional Features

* QR Code Generation
* Responsive UI
* Modern Dashboard Design

## Tech Stack

### Frontend

* React.js
* JavaScript (ES6+)
* CSS3
* Axios
* React Router
